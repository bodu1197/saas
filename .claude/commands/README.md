# Claude Code 슬래시 커맨드

이 디렉토리에 `*.md` 파일을 두면 Claude Code에서 `/파일명`으로 호출할 수 있습니다.

## 현재 등록된 커맨드

| 커맨드      | 파일          | 용도                                           |
| ----------- | ------------- | ---------------------------------------------- |
| `/시작`     | `시작.md`     | 작업 세션 시작 전 필수 점검                    |
| `/회고`     | `회고.md`     | 작업 종료 시 자동 회고 + 교훈 갱신             |
| `/교훈추가` | `교훈추가.md` | 새로운 실수를 LESSONS.md에 기록                |
| `/리뷰5`    | `리뷰5.md`    | 5단계 코드 리뷰 (타입→보안→품질→아키텍처→교훈) |
